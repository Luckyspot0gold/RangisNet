## üéì Understanding DNS Concepts

### What is DNS?
Domain Name System (DNS) translates domain names (rangis.net) to IP addresses that computers can understand.

### What are Nameservers?
Nameservers are DNS servers that store and manage DNS records for your domain.

### What is SSL?
SSL (Secure Sockets Layer) certificates enable HTTPS, encrypting data between users and your server.

### Vercel DNS vs Cloudflare DNS
- **Vercel DNS:** Managed by Vercel, automatic setup, integrated with deployment
- **Cloudflare DNS:** Managed by Cloudflare, manual setup, advanced features

For detailed explanations, see [DNS_CONFIGURATION_GUIDE.md](./DNS_CONFIGURATION_GUIDE.md)
