## üé• PRODUCTION SPECIFICATIONS

### **Technical Requirements:**

**Video Format:**
- Resolution: 1920x1080 (1080p)
- Frame Rate: 30fps (or 60fps if you can)
- Codec: H.264
- Bitrate: 8-10 Mbps (high quality)
- Audio: 256 kbps AAC stereo

**Recording Tools:**
- **macOS:** QuickTime Player (Cmd+Ctrl+5)
- **Windows:** OBS Studio (free) or Xbox Game Bar
- **Linux:** OBS Studio or SimpleScreenRecorder
- **Cloud:** Loom (loom.com) - easier editing

**Screen Capture Settings:**
- Capture full screen OR specific window
- Enable microphone audio
- Disable system notifications (Do Not Disturb)
- Close unnecessary browser tabs
- Use Incognito/Private mode to hide bookmarks

### **Audio Recording:**

**Microphone Setup:**
- Use headset mic (better than laptop mic)
- Record in quiet room (no background noise)
- Test audio levels before full recording
- Speak clearly, moderate pace
- Pause between major sections (easier editing)

**Audio Tips:**
- Record voiceover SEPARATELY if needed
- Use Audacity (free) to clean up audio
- Remove background hiss: Effect ‚Üí Noise Reduction
- Normalize volume: Effect ‚Üí Normalize (-3dB peak)
